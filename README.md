el proyecto tiene como objetivo poder analizar el rendimiento de productos en una empresa de ventas, utilizando sql para lograr identificar ingreso, rentabilidad y oportunidades de mejora.
las herramientas que se utilizaron:
postgreSQL
SQL
pgAdmin
en el analisis se logreo identificar que el zapato negro era el producto que genera mayores ingresos debido a su alto precio
el producto con mayor rentabilidad es la camisa naranja, destaca como un producto muy rentable ya que convina buen margen de ganancia y volumen de ventas
la sudadera gris presenta un bajo rendimiento tanto como en ventas como en ganancia, este producto se considera muy ineficiente
se recomienda priorizar productos con alta rentabilidad y demanda
reducir inventario de productos con bajo rendimiento
evaluar estrategias de precios y expansión del mercado
